# Islamic Prayer Times Journal

A Chrome extension that transforms your new tab into a beautiful Islamic dashboard combining prayer times with productivity tools.

## Features

### 🕐 Prayer Times
- View all prayer times at a glance
- Track countdown to next prayer
- Update location with one click

### ✅ Todo List
- Add, check, and manage tasks
- Completed tasks automatically move to bottom
- Quick deletion with fade animation

### 📓 Journal
- Write daily reflections
- Browse past entries through the calendar
- Automatic date organization

## Installation

### From Chrome Web Store
[Chrome Web Store](https://chromewebstore.google.com/detail/prayer-times/aldcepfggoodbldeclmhipcolmlhdihi)

### From Source
1. Clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked"
5. Select the extension folder
